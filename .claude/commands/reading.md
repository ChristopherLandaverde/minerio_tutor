# Brazilian Portuguese Reading Comprehension Practice

Practice reading and understanding Brazilian Portuguese texts at A2 level, with Mineiro flavor.

## Protocol

### 1. Greet and Explain

Greet the learner: "Oi {name}! E aí, tudo bem?" Explain we're doing reading comprehension, 15–20 min. Remind them: read the whole text first, don't translate every word, use context clues.

### 2. Select Text Type

Choose based on interests and level. A2 types: personal emails, short news, ads, instructions/recipes, simple stories, blog posts, social media, event leaflets. Length: 100–200 words.

### 3. Present Text

Show the text with topic, type, and word count. Tell the learner to type "ready" when done reading.

### 4. Comprehension Questions (ONE AT A TIME!)

Present 5 questions, one at a time, waiting for each answer before showing the next:

1. **Pergunta — Ideia Principal:** "Sobre o que é o texto?" with multiple choice (a/b/c)
2. **Pergunta — Detalhes:** Specific question about a detail in the text
3. **Pergunta — Vocabulário:** "No texto aparece '{word}'. O que isso significa?" with multiple choice
4. **Pergunta — Compreensão:** Inference question — learner answers in Portuguese
5. **Pergunta — Verdadeiro ou falso?** Statement about the text

### 5. Feedback After Each Question

Show correct/incorrect, the answer, explanation, **Score: X/10**, and if wrong quote the relevant text: "No texto diz: '...'"

### 6. Vocabulary Review

Show table of new words (Português | English | Exemplo do texto). Ask if the learner wants to save them to spaced repetition. If yes, add to vocabulary database.

### 7. Session Summary

Show: text topic, word count, number of questions, accuracy %. Break down by category (main idea ✅/❌, details score, vocabulary score, inference score). List new words learned, reading strategies used well, tips for next time.

## Example A2 Texts

### Example 1: Personal Email

```
Oi Chris,

Tudo bom? Que saudade! Soube que você tá pensando em vir pra Belo Horizonte mês que vem. Que demais, uai! Eu tô livre no sábado, dia 15. A gente pode se encontrar lá no Mercado Central, umas 14h. Dá pra tomar um cafezinho e depois andar pelo centro da cidade.

O tempo aqui em março é quente, mas chove bastante à tarde, então traz um guarda-chuva, tá? Tô doida pra te ver!

Beijos,
Ana
```

### Example 3: Simple News

```
NOVA LINHA DE METRÔ EM BELO HORIZONTE

Belo Horizonte vai ganhar uma nova linha de metrô este ano. As obras começam em abril e devem durar até o final do ano.

"Queremos que o transporte público seja mais fácil pra todo mundo," disse o prefeito. A nova linha vai passar por bairros onde moram muitas famílias.

Durante as obras, algumas ruas podem ficar fechadas. Os moradores devem usar rotas alternativas. Mais informações no site da prefeitura.
```

## Reading Progression

**A2 Early:** Personal emails (100 words), simple ads/menus, basic instructions and recipes

**A2 Advanced:** Short news articles (200 words), blog posts about daily life, event leaflets

**Moving to B1:** Longer articles (300+ words), opinion pieces and crônicas, complex instructions, stories with Mineiro expressions
